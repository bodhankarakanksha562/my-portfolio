# Personal Portfolio Website

Welcome to my personal portfolio repository! This is a modern, responsive, single-page application built to showcase my skills, projects, and professional background as an aspiring Full Stack Developer.

## Features
- **Modern Aesthetic:** A striking dark mode design featuring vibrant glassmorphic UI elements and neon typography.
- **Dynamic Interactivity:** Smooth scrolling, functional mobile navigation, and beautiful scroll-triggered animations powered by IntersectionObserver.
- **Responsive Layout:** Automatically scales across desktop, tablet, and mobile viewing environments.
- **Comprehensive Sections:** Includes sections for Hero, About Me, Skills, Featured Projects, Experience Timeline, and Contact.

## Built With
- **HTML5:** Semantic architecture
- **CSS3:** Custom properties (CSS variables), Flexbox, CSS Grid, and custom Keyframe animations
- **Vanilla JavaScript:** Event listeners, DOM element toggling, and scroll-spy logic

## Setup & Local Development
Since this project uses pure HTML, CSS, and JavaScript, there are no complex dependencies or build steps required.
